# 数据分析师 (Data Analyst)

## 功能概述

本技能专注于数据处理与分析：

1. **数据读取** - 支持 Excel(.xlsx/.xls)、CSV、JSON 格式
2. **数据清洗** - 去除重复、填充缺失值、格式转换
3. **统计分析** - 计算均值、中位数、增长率、环比等指标
4. **数据可视化** - 生成柱状图、折线图、饼图等
5. **报告生成** - 整合分析结果，生成结构化报告

## 使用场景

- 用户需要处理 Excel 表格数据
- 需要从大量数据中提取关键指标
- 需要生成数据可视化图表
- 需要将数据整理成报告

## 工作流程

```
1. 确认数据源文件路径
2. 读取并解析数据
3. 数据清洗与预处理
4. 计算关键指标
5. 生成图表（如需要）
6. 输出分析报告
```

## Python 依赖

使用 `pandas` 进行数据处理，`matplotlib` 进行可视化：

```python
import pandas as pd
import matplotlib.pyplot as plt

# 读取数据
df = pd.read_excel('file.xlsx')

# 数据清洗
df = df.drop_duplicates()
df = df.fillna(0)

# 计算指标
total = df['column'].sum()
avg = df['column'].mean()

# 生成图表
```

## 常用分析模板

### 销售数据分析
- 销售额总计/平均值
- 各类别销售占比
- 环比/同比增长
- TOP10 产品

### 客户数据分析
- 新增客户数
- 客户留存率
- 客单价分布
- 客户等级分布

### 运营数据分析
- 转化率
- 活跃度
- 复购率
- 响应时间
