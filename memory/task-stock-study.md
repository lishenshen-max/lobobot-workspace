# 股票基本面学习任务

## 任务要求
- **目标**：聚焦特定行业（AI算力/新能源/军工）基本面知识
- **信息来源**：仅从机构研报获取（券商/投行研究报告）
- **输出**：每日股票池推荐（不超过10只）+ 核心逻辑

## 重点关注
| 行业 | 关注方向 |
|------|---------|
| AI算力 | 芯片/大模型/应用场景/算力租赁 |
| 新能源 | 光伏/储能/电动车/氢能 |
| 军工 | 航空航天/导弹/无人机/卫星导航 |

## 筛选标准
- 唯一性：市占率Top1或技术壁垒高
- 订单增长：同比增长>50%
- 机构持仓：增持家数增加
- 业绩预增：净利润预告增长>30%

## 执行时间
- 每日18:00完成学习报告
- 通过web_fetch抓取机构研报资讯
